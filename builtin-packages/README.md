# Builtin Standard APK

本目录用于放置内置保底标准包。

默认文件名：

- `standard.apk`

默认路径：

- `builtin-packages/standard.apk`

可通过环境变量覆盖：

- `BUILTIN_STANDARD_APK_PATH`

## 作用

当以下情况发生时，系统会尝试回退到这里的内置包：

- 标准包配置为空
- 当前激活标准包失效
- 部署环境希望提供一个固定保底 APK

## 使用建议

- 本地开发时可以不放
- 如果你希望部署后标准包功能开箱即用，建议在镜像构建或部署挂载时提供该文件
- 仓库默认会忽略 `*.apk`，不要指望 Git 跟踪实际 APK 二进制

## 与标准包配置的关系

优先级通常是：

1. 当前配置的标准包
2. 上一次有效标准包
3. 本目录内置保底标准包
